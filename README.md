# AMON Corporate Website

This is a premium, Apple-style corporate one-page website for AMON built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

## Setup Instructions

Since the project was scaffolded manually in an environment without Node.js, you need to install dependencies first.

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push this repository to GitHub.
2. Link the repository in Vercel.
3. Vercel will automatically detect `Next.js` and deploy.

## Features

- **Apple-like Aesthetic**: Snow white matte background, glassmorphism, and subtle animations.
- **Interactive Elements**: Magnetic buttons, mouse aura effect in Hero.
- **Responsive**: Fully optimized for Mobile, Tablet, and Desktop.
- **Privacy-First**: No external analytics or tracking scripts included by default.

## Project Structure

- `/app`: Main application routes and layout.
- `/components/ui`: Reusable primitives (Buttons, Cards).
- `/components/sections`: Landing page blocks (Hero, Roadmap, etc.).
- `/lib`: Animation variants and utility functions.
